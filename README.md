Data Explorer – Project Overview
Data Explorer is a web-based application designed for seamless CSV file management and data analysis. The primary goal is to provide users with an intuitive interface to upload, store, and explore datasets directly in a browser.

Key Features:
CSV Upload & Storage: Securely upload and store datasets in a relational database.

Data Preview: Instant tabular view of uploaded data.

Automated Statistics: Real-time calculation of basic metrics (Min, Max, Average) for numeric columns.

Future Roadmap: Integration of interactive charts for enhanced data visualization.

Tech Stack:
We chose Django (Python) as the core framework. Its robust MVT (Model-View-Template) architecture ensures a clean separation of concerns, allowing for scalable backend logic, efficient database management, and a responsive user interface.
